# 🎤 Voice Form Filling - Quick Start

Get started with voice-powered form filling in 30 seconds!

## The 30-Second Demo

1. **Start the app**
   ```bash
   # Make sure both backend and frontend are running
   ```

2. **Go to any form**
   - Click Dashboard → Any service (e.g., "Income Certificate")
   - OR: Click a scheme from search results

3. **Watch for the green "Listening..." badge**
   - It appears automatically in the bottom-right corner
   - Means continuous voice mode is active

4. **Fill the form with your voice**
   ```
   Click "Name" field → Say: "Your Name"
   (Automatically moves to Age)
   Say: "Your Age" (e.g., "25")
   (Automatically moves to Aadhar)
   Say: "Your 12-digit Aadhar" (e.g., "1234 5678 9012")
   (Automatically moves to Address)
   Say: "Your Address"
   Done! ✓
   ```

5. **Watch the magic happen**
   - Fields turn **blue** when you focus them
   - **Gray preview text** appears as you speak (interim)
   - Fields turn **green** with a ✓ when filled
   - Auto-moves to next field after each fill

## Two Ways to Use Voice

### Method 1: Focus & Speak (EASIEST - Zero thinking required)

```
1. Click or tab into any field
2. Just say the value
3. That's it!
```

**Example:**
- *Click* Name field → *Say* "Amit Sharma" → ✓
- *Click* Age field → *Say* "thirty two" → ✓
- *Click* Aadhar field → *Say* "1234 5678 9012" → ✓

### Method 2: Voice Commands (Hands-free)

```
Just speak commands without clicking:
- "My name is Amit Sharma"
- "I am 32 years old"
- "Aadhar number is 1234 5678 9012"
```

## Visual Guide

### What You'll See

1. **Green Badge (Top Right)**
   ```
   🟢 Listening...
   Filling: name
   ```
   Shows continuous mode is active and which field is focused

2. **Blue Field Border**
   - Field is focused and ready for voice input

3. **Gray Italic Text** (in field)
   - Real-time preview of what you're saying
   - Updates as you speak

4. **Green Border + ✓ Checkmark**
   - Field successfully filled by voice
   - Auto-disappears after 2 seconds

5. **Blue Speech Bubble**
   ```
   "amit sharma"
   ```
   Shows interim transcript while you're speaking

6. **White Confirmation**
   ```
   You: Amit Sharma
   ✓ name filled
   ```
   Confirms what was heard and which field was filled

## Controls

### ON/OFF Button
- **Green button with "ON"** = Continuous mode active
- **Gray button with "OFF"** = Continuous mode disabled
- Click to toggle

### Guide Mode
- Say **"Guide me"** or **"Help"** to open step-by-step instructions
- Say **"Next"** / **"Previous"** to navigate steps
- Say **"Stop"** to close guide

## Tips for Perfect Results

### ✓ Do This
- Speak **clearly** at normal pace
- **Click the field** first for guaranteed accuracy
- Use Chrome or Edge browser
- Allow microphone permissions

### ✗ Avoid This
- Speaking too fast or mumbling
- Background music/noise during input
- Using Firefox/Safari (limited support)
- Denying microphone access

## Troubleshooting

### Problem: Nothing happens when I speak
**Solution**: Check if green "Listening..." badge is visible. If not, click the ON/OFF button.

### Problem: Wrong field gets filled
**Solution**: Click/focus the correct field first, then speak.

### Problem: "Microphone permission denied"
**Solution**: 
1. Click the 🔒 or microphone icon in browser address bar
2. Select "Always allow"
3. Refresh the page

### Problem: Interim text looks wrong
**Solution**: Wait! Interim is just a preview. Final result is usually more accurate.
